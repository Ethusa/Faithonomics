# Faithonomics LMS Curriculum Structure

The LMS maps the Faithonomics platform as:

- Course: `Faithonomics Economic Discipleship`
- Level: Wix CMS `Modules`
- Session: Wix CMS `Lessons`
- Step: structured `Lesson.content` blocks

Each session starts with four core steps:

1. Core Teaching
2. Scripture and Principle
3. Business Application
4. Reflection Checkpoint

Lecturers can add extra required or optional steps for Wix Media video, images, downloads, web links, and secure custom HTML activities. Required steps are included in the completion gate with quizzes and group discussions before the next session unlocks.
